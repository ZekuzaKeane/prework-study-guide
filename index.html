Prework Study Guide

HTML

  • DOCTYPE declares to the browser what kind of document to expect when opened.
  • The HTML element represents the root of the HTML document.
  • The head element contains the metadata for the web application.
  • The title element defines the title for the web page.
  • The style element defines the CSS styles associated with the web page.
  • The base metadata tag defines the base URL for the webpage.
  • The link metadata tag connects an external resource to the HTML document.
  • A metadata tag defines metadata such as character set, description, keywords, author and more.
  • The script metadata tag embeds client-side scripts in the HTML document.
  • H1/H2/H3/H4/H5 are basic headers of varying sizes.
  • UL, OL, and LI are unordered list, ordered list and list item respectively.

CSS

  • Selectors define the element or attribute to which the CSS rules apply.
  • The Margin CSS property acts like a traditional word processing margin. It is not visible on the page.
  • The Padding CSS property adds space around the content insane an element.
  • The Width CSS property changes the width of the element.
  • The Height CSS property changes the height of the element.
  • The Background Color CSS property changes the background color of the element.
  • The Color CSS property changes the color of the text.
  • The Text Align CSS property changes the text alignment in an element.
  • The Font Size CSS property changes the size of the font.
  • The Border CSS property assigns a border and border color to the assigned element.
  • The Box Shadow CSS property draws a shadow effect around the element. The first value defines the X offset, the second defines the Y offset, and the last value defines the color.
  • The .class selector designates a class, which gives a property assignment to all elements with that class.

Git

  • git status checks what branch the project is currently on.
  • git checkout -b branch-name creates a new branch and switches to it.
  • git add -A commits all files in the working branch.
  • git commit -m "comment" associates a message with the commit.
  • git pull origin main syncs the local branch with the remote branch.
  • git push origin "branch" pushes all changes.
